Double Calendar Spread — Strategy Builder

Sell a near-term call + put (a little OTM each side), buy the same two strikes in a far-term expiry. Net DEBIT, defined risk (max loss = debit paid), long vega / positive theta. Profit is maximized if price sits between the two short strikes ("sweet spot") when the near leg expires. This tool auto-picks both expiries and both strikes for you — override either if you want manual control.
